## üõ†Ô∏è Setup Instructions for Gumroad

### Step-by-Step Product Creation

1. **Create Each Product in Gumroad**
   - Log into Gumroad
   - Click "New Product"
   - Select "Membership" as product type
   - Choose "Monthly" recurring payment

2. **Fill Product Details**
   - Copy the product name from this document
   - Paste the full description
   - Set the price
   - Add product image (logo/screenshot)

3. **Add Custom Field**
   - Go to product settings
   - Navigate to "Custom Fields"
   - Add field: `user_id` (Hidden, Required)

4. **Get Permalink**
   - After creating product, note the permalink
   - Update your `.env.local`:
     ```
     NEXT_PUBLIC_GUMROAD_STARTER_PERMALINK=crewquo-starter
     NEXT_PUBLIC_GUMROAD_PRO_PERMALINK=crewquo-professional
     NEXT_PUBLIC_GUMROAD_ENTERPRISE_PERMALINK=crewquo-enterprise
     ```

5. **Test Purchase Flow**
   - Use Gumroad test mode
   - Complete test purchase
   - Verify webhook receives data
   - Check Firestore updates
